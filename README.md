
## Image-Gallery

Simple image gallary with beutifull UI created using Next JS

[Click Here to see Live Website](https://image-gallery-updated.vercel.app/)  

## Features

* Beutifull UI
* LightBox Feature
* Pagination Feature
* Search Functionality
* Responsive design
