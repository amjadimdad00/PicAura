## PicAura 📸

Simple image gallery with beautiful UI created using Next JS ✨

[Click Here to see Live Website 🌐](https://picaura.vercel.app/)  

## Features 🚀

* Beautiful UI 🌟
* LightBox Feature 🖼️
* Pagination Feature 📄
* Search Functionality 🔍
* Responsive design 📱
